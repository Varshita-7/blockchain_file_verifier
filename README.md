# Blockchain File Verifier
A web application built with **Python** and **Streamlit** that verifies file integrity using **SHA-256 hashing** and blockchain concepts.

## Features
- File upload and verification
- SHA-256 hash generation
- Tamper detection
- User-friendly interface

## Technologies
- Python
- Streamlit
- SHA-256
- Blockchain
